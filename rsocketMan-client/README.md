
# rsocketMan 🐳

A devtool for rsocket protocol 🔧

## Installation 🌟

In rsocketMan-server and in rsocketMan-client

```bash
  npm install
```

## Usage 🍉

- In rsocketMan-server run `ts-node index.ts`


- In rsocketMan-client run `npm run dev `
